# Generelle Anforderungen
- grundsätzliche Beschreibung der App, was soll sie können, was ist das Ziel, etc.
- multi language support (deutsch, englisch, französisch, chinesisch, russisch, indisch, arabisch, spanisch, portugiesisch, italienisch)
- offline support (die App soll auch ohne Internetverbindung funktionieren, z.B. durch Caching von Informationen und Funktionen)
- die app läuft auf einem android tablet im querformat, welches in der Wohnung zur Verfügung steht. 
- Externe Links sollen nicht geöffnet werden, sondern als QR Code angezeigt werden, damit die Gäste sie mit ihrem eigenen Smartphone scannen können.
- Alle Texte & Links, etc. sollen über Notion Db's konfigurierbar sein, damit sie einfach angepasst werden können, ohne dass der Code geändert werden muss

## Widgets

### Wetter
- Aktuelle Wetterbedingungen und Vorhersage für die nächsten 3 Tage, damit die Gäste ihre Aktivitäten planen können.

### Was willst du heute unternehmen?
- Eine Übersicht über die verschiedenen Aktivitäten und Angebote in der Umgebung, die Gäste unternehmen können. ... 

### Balkon Beschallung
- Möglichkeit, vordefinierte Audioinhalte auf dem Balkon abzuspielen, um eine angenehme Atmosphäre zu schaffen. Z.B. Vogelgezwitscher, Meeresrauschen, Waldgeräusche, etc.
- Hinweis, dass die Device auf dem Balkon eingeschaltet sein muss, damit die Beschallung funktioniert.

### ÖV
- Aktuelle ZVV Störungen, die Manegg betreffen, werden in Echtzeit angezeigt, damit die Gäste über mögliche Verzögerungen oder Änderungen im öffentlichen Verkehr informiert sind.
